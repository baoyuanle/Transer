# Transer 

## 简介
golang 翻译器

目前实现了百度和有道的翻译API对接

提供了两个命令行工具:

## Transer

单条翻译，有道 只支持13国语言。

![](Preview/Transer.png)



## QtTranser

Qt的多语言文件批量翻译

![](Preview/QtTranser.png)
